# 思播 SiMedia

**思源笔记的多源媒体播放器 · 网盘媒体中心 · 时间戳笔记工作台**

让思源笔记同时成为统一播放器、网盘媒体浏览器、时间戳记录器与学习型媒体工作区。  
支持本地媒体、BBLL、TTVV、OpenList、WebDAV、百度网盘、阿里云盘、夸克、115、123 网盘、OneDrive、S3 等多种来源。
